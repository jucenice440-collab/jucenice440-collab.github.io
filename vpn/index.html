搭建你自己的 WireGuard VPN

家庭静态独享海外IP · 年费不到¥100 · 一条命令搞定

为什么选择自建 VPN？

❌ 共享机场

几十人挤一个IP
随时被封 · 速度不稳
被网站识别为代理

✅ 自建 VPS

家庭静态独享IP
仅你一人使用
不被墙识别 · 稳定高速

核心卖点：自己搭建的 VPN 给你一个家庭静态独享 IP 地址，全世界只有你一个人在使用。不会被识别为机场节点，速度和稳定性都有保障。

三步搭建

  1. 买一台海外 VPS
    推荐 aaitr.com 静态家宽 VPS，最低配 $10/年。
    系统选 Ubuntu 22.04 或 Debian 12。拿到 IP、用户名(root)、密码。
  2. 一键安装 WireGuard
    下载 FinalShell 连上 VPS，然后复制下面命令： 
    curl -L https://raw.githubusercontent.com/angristan/wireguard-install/master/wireguard-install.sh | bash
    按提示填：客户端名 iphone，端口 51820，DNS 选 Cloudflare。不到两分钟装好。
  3. 手机电脑扫码连接
    iPhone 装 WireGuard App，扫二维码导入配置。
    Windows/Mac 装客户端，导入配置文件。打开开关即连通。

防火墙配置

# 开放 WireGuard 端口
ufw allow 51820/udp
ufw allow 22/tcp
ufw enable

# 验证运行状态
systemctl status wg-quick@wg0

iPhone 配置（二维码方式）

# 生成二维码
apt install qrencode -y
qrencode -t ansiutf8 < /root/iphone.conf

手机打开 WireGuard App → 添加隧道 → 扫描二维码 → 连接。看到 VPN 图标即成功。

验收

  • ✅ 开 VPN → 访问 google.com → 能打开
  • ✅ 关 VPN → 访问不了 → 确认走代理
  • ✅ YouTube 播放 4K 视频不卡顿
  • ✅ 测速 fast.com 延迟 < 80ms
